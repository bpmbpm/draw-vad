# Test Cases - ARIS Express Clone

Данный документ описывает тестовые сценарии (Test Cases) для проверки функциональности приложения ARIS Express Clone.

## Окружение тестирования

- **URL приложения**: https://bpmbpm.github.io/draw-vad/aris/ver1/index.html
- **Примеры диаграмм**: папка `examples/`
- **Трафареты**: папка `stencils/`

---

## Test Case 1: Выбор и просмотр примера VAD диаграммы

### Описание
Проверка загрузки и отображения примера VAD (Value Added Diagram) диаграммы из Проводника моделей.

### Предусловия
1. Открыть приложение в браузере: https://bpmbpm.github.io/draw-vad/aris/ver1/index.html
2. Убедиться, что приложение загрузилось (статус "Готов к работе...")

### Шаги выполнения
1. В левой панели "Проводник моделей" найти раздел "Примеры"
2. Кликнуть на "VAD Пример 1 - Управление заказами"
3. Дождаться загрузки диаграммы

### Ожидаемый результат
- Заголовок холста изменится на "VAD Пример 1 - Цепочка создания ценности"
- На холсте отобразятся элементы диаграммы (стрелки-функции, связи)
- В статусной строке появится сообщение "Загружен пример: ..."

### Файлы примеров
- `examples/vad_example_1.drawio`
- `examples/vad_example_2.drawio`

## Test Case 3: Работа с трафаретами (Stencils)

### Описание
Проверка загрузки и использования трафаретов для создания элементов диаграмм.

### Предусловия
1. Открыть приложение в браузере
2. Создать новую диаграмму: Модель → Новая VAD диаграмма

### Шаги выполнения
1. В левой панели переключиться на вкладку "Трафареты"
2. В выпадающем списке выбрать "VAD - Value Added Diagram"
3. Убедиться, что трафареты загрузились (отображаются элементы)
4. Кликнуть на элемент трафарета "Функция (Function)"

### Ожидаемый результат
- Список трафаретов отображается без ошибок
- При клике на трафарет элемент добавляется на холст
- При перетаскивании (drag-and-drop) элемент добавляется в указанную позицию

### Файлы трафаретов
- `stencils/vad.xml` - VAD трафареты
- `stencils/epc.xml` - EPC трафареты
- `stencils/bpmn.xml` - BPMN трафареты
- `stencils/org.xml` - Организационные трафареты

## Test Case 4: Модификация и сохранение диаграммы

### Описание
Проверка возможности изменения диаграммы и её сохранения в файл.

### Предусловия
1. Открыть приложение в браузере
2. Загрузить пример диаграммы (например, VAD Пример 1)

### Шаги выполнения
1. Добавить новый элемент из трафаретов на холст
2. Выбрать меню Файл → Сохранить как...
3. Ввести имя файла (например, "my_diagram")
4. Подтвердить сохранение

### Ожидаемый результат
- Браузер предложит скачать файл с расширением `.drawio`
- Файл содержит корректный XML в формате draw.io
- Файл можно открыть в draw.io или импортировать обратно в приложение

## Test Case 7: Переключение между нотациями в трафаретах

### Описание
Проверка переключения между различными типами нотаций в панели трафаретов.

### Шаги выполнения
1. Переключиться на вкладку "Трафареты"
2. Выбрать "VAD - Value Added Diagram" → проверить загрузку трафаретов
3. Выбрать "EPC - Event-driven Process Chain" → проверить загрузку трафаретов
4. Выбрать "BPMN" → проверить загрузку трафаретов
5. Выбрать "Организационная структура" → проверить загрузку трафаретов

### Ожидаемый результат
- Каждая нотация загружает свои уникальные трафареты
- Нет ошибок в консоли браузера
- Все элементы трафаретов отображаются с иконками и названиями

## Известные ограничения

1. Полноценный draw.io редактор не интегрирован - используется SVG-рендеринг
2. Drag-and-drop работает в базовом режиме
3. Связи между элементами отображаются упрощённо
